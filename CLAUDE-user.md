# Global Claude Code Behavioral Rules

Always-loaded rules: [`./rules/`](./rules/)

## Writing Standards

**Semicolons**: Use to join independent clauses or separate complex list items. Never chain explanation steps with semicolons (e.g., "X does this; Y handles that; Z provides the other").

**Dashes**: Use ` – ` (space-en-dash-space) for asides and breaks. Em-dashes only in attribution lines or interrupted dialogue.
